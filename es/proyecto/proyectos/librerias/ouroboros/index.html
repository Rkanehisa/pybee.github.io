Ouroboros

Una implementación de la biblioteca estándar de Python, escrita en Python. Esto permite que las librerías estándar de Python se puedan utilicen de forma portátil en cualquier implementación de la máquina virtual Python, es decir, no requiere CPython.

Ampliación de las pruebas unitarias.

Entonces, ¿por qué se llama "Ouroboros"?

Ouroboros es el símbolo antiguo que representa a una serpiente que come su propia cola. Ouroboros (esta librería) también es similarmente autorreferencial; implementa Python en Python (el subconjunto de Python en la máquina virtual Python).
