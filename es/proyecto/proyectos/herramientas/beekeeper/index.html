BeeKeeper

Herramienta de construcción y de integración continua

BeeKeeper es una herramienta para ejecutar contenedores Docker bajo demanda, con reportes de estado de GitHub. Como efecto secundario, esto lo hace muy efectivo como una herramienta de revisión de código automatizada, un sistema de integración continua, un sistema de despliegue automatizado y muchas otras cosas.

Entonces, ¿por qué se llama "BeeKeeper"?

Alguien tiene que cuidar la salud y el bienestar de todas las abejas...
