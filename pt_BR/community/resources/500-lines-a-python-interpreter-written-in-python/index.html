500 Linhas: Um interpretador Python escrito em Python

by Allison Kaptur; published Julho 12, 2016

Byterun é um interpretador Python implementado em Python. Graças ao meu trabalho no Byterun, fiquei supresa e feliz em descobrir que a estrutura básica do Python cabia facilmente dentro da restrição de tamanho de 500 linhas. Este capítulo vai mostrar a estrutura do interpretador e lhe dar base suficiente para continuar explorando. O objetivo não é explicar tudo que há a ser aprendido sobre interpretadores — como tantas outras áreas de interesse na programação e ciência da computação, você precisa dedicar muitos anos para desenvolver um conhecimento profundo sobre o tópico.
